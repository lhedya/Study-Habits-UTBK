# Sistem Rekomendasi Belajar UTBK

Aplikasi berbasis AI untuk memberikan rekomendasi belajar yang dipersonalisasi untuk persiapan UTBK.

## Features

- Profiling 8 tipe learner berdasarkan survey 52 pertanyaan
- Analisis 5 dimensi belajar (Gaya Belajar, Konsistensi, Manajemen Waktu, Kesiapan UTBK, Kondisi Psikologis)
- Gap analysis antara skor saat ini dan target
- Rekomendasi belajar yang dipersonalisasi
- Study plan mingguan
- Milestone timeline 4 bulan

## Live Demo

(https://study-habits-utbk-8apllsedzrcbbabe85ezn6.streamlit.app/)

## Tech Stack

- **Frontend:** Streamlit
- **ML Models:** Scikit-learn (K-Means, Random Forest)
- **Visualization:** Plotly
- **Data Processing:** Pandas, NumPy

## Model Performance

- K-Means Clustering: Silhouette Score ~0.6
- Random Forest Classifier: Accuracy >80%
